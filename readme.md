# suite-laravel

## git clone
```shell
git clone https://github.com/absuite/suite-laravel.git
```

## install package
```shell
cd suite-laravel

rename composer.json.prod composer.json composer.json.prod

composer install

```


## init project
```shell
composer run-script gmf-install
```

## install data
```shell
php artisan gmf:install --force
```
